Starting with version 1.0.0.1, the buttons are configurable for each model. More information can be found in the wiki.
Softswitch for Benedini
The softswitch can simulate the **1st Coder 12-Key** for the PROP3.
As an add-on, you can also configure the **2nd Coder 2-Key** for the PROP2.

This allows up to 24 functions to be controlled via touch.
[Benötigt EdgeTX 2.8.x oder höher](https://github.com/EdgeTX/edgetx)
